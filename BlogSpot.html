Registro de lo aprendido en el curso de GIT

La idea de este blog es ir registrando los aprendizajes del curso de Git en Platzi.

Comandos Linux

  • ls:Lista los archivos y carpetas en la ruta actual.
  • cd:Permite navegar entre carpetas. 'cd "Nombre de Carpeta"' permite ingresar a esa carpeta si existe en la ruta actual. 'cd ..' permite regresar a la carpeta anterior. Tambien se puede usar ruta completa. 'cd c:/ruta1/carpeta1/'
  • clear:Borra la pantalla, despeja la consola
  • mkdir:crea una carpeta en la ruta actual
  • rm:Borra archivo o carpeta. Poner especial cuidado pues este comando mal usado puede dañar el SO.
  • cat:permite ver el contenido de un archivo en consola
  • pwd:muestra la ruta completa en la que nos encontramos actualmente
  • --help:complemento para ayuda de cada comando. Ejemplo: 'git --help' muestra la ayuda del comando git en consola

Comandos Principales de Git

  • git: Palabra reservada y comando principal, con el le decimos a la consola que deseamos ejecutar alguna acción en git.
  • git init:Comando para inicializar el repositorio, se debe navegar hasta la ruta especifica que deseamos como repositorio principal y ejecutar este comando.
